## Wetter-App

Die App zeigt das Wetter für Städte an. Man kann eine Stadt eingeben und bekommt:

- Aktuelle Temperatur
- Gefühlt Temperatur
- Wettersymbol

## Nutzung

1. Repository klonen oder ZIP herunterladen
2. `index.html` im Browser öffnen

## Verwendet

- HTML
- CSS
- JavaScript
- OpenWeatherMap API
